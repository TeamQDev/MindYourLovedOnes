How to Import Documents (Android)

Via email: (more detailed instructions are available in FAQ’s)

- Send an email to the iPhone address and attach your document(s)

- Open email to see document file attachment with paperclip showing on left side. Based on which version of Android OS is being used there are 2 ways to save documents to Download folder:

1) click on 3 dots to the right and ‘Save’ to Download folder This option appears in all OS versions

or

2) click on image and ‘Save’ document to Download folder

- The attached document by default is downloaded to the SD card in the download folder in the files section of the storage space on disk

- Once you clicked the Download button, the attached file will render in the compatible 3rd party application, in most cases Adobe Reader is the default

- Go back to My Care Wishes app, go the Dashboard menu with the 2 or 4 large square boxes and go to your Advance Care Plan Documents section and click on the ‘Add’ Button top of the screen

- Add button will go to the file section on your Android where the document(s) reside, which in most cases is the “Download’ folder/directory

- Click on the document you wish to add and it will be added to your document section in My Care Wishes App

Using DropBox: (more detailed instructions are available in FAQ’s)

- Tap on the “Add” button on top of screen, two options will be available to import documents

1) “SD card” (location of documents in “Downloads” folder)

2) “DropBox”

When user selects the DropBox option, a screen appears with login button for user to sync DropBox account with My Care Wishes app

- When user selects the DropBox option, a screen appears with login button for user to sync DropBox account with My Care Wishes app.

- Documents appear on the “Document List in DropBox” screen after downloading. (if documents do not appear, login to DropBox)

- From “Document List in DropBox” screen select the document(s) to be added and tap the save button
